## Requirements
---------------
Requirements untuk menjalankan program ini adalah komputer telah terpasang gcc.


## Cara Menggunakan Program
---------------------------
1. Membuka executable file bernama t
    Alternatif jika tidak bisa :
        buka terminal di folder IF2110-11-02
        ```
        >> ./t
        ```
    Alternatif lain :
        buka terminal di folder dan satukan semua modul
        ```
        >> gcc unnamed.c matriks.c mesinkar.c map.c bintree.c listrek.c asd.c stackt.c queue.c array.c graph.c point.c jam.c -o t
        >> ./t
        ```
